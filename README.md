# Dashboard de Monitoramento de Incidentes de Infraestrutura

## Sobre o Projeto

Este projeto simula um fluxo completo de Análise de Dados aplicado ao
contexto de Infraestrutura e Cibersegurança.

### Pipeline

``` text
CSV
↓
Python (Pandas)
↓
ETL
↓
PostgreSQL
↓
SQL
↓
Power BI
```

## Tecnologias

-   Python
-   Pandas
-   PostgreSQL
-   SQL
-   Power BI
-   Git/GitHub

## ETL

-   Tratamento de valores nulos
-   Padronização de datas
-   Correção de inconsistências
-   Ajuste de tipos de dados
-   Preparação para PostgreSQL

## Análises SQL

-   Total de incidentes
-   Incidentes por servidor
-   Incidentes por categoria
-   Incidentes críticos
-   Tempo médio de resolução
-   Média de CPU por servidor

## Dashboard

-   Total de Incidentes
-   \% de Incidentes Críticos
-   CPU \> 80%
-   Memória \> 85%
-   Disco \> 90%
-   Incidentes por mês
-   Tempo médio por equipe
-   Incidentes por severidade
-   Incidentes por servidor

## Objetivo

Demonstrar um pipeline completo de dados, desde a preparação dos dados
até a construção de um dashboard executivo.

## Autor

Projeto desenvolvido para portfólio de Analista de Dados.
